# Calculatrice graphique en Ruby

Cette application est une calculatrice simple développée en Ruby avec Tk.

Le projet aide à comprendre :

- les opérations mathématiques de base ;
- les interfaces graphiques ;
- les événements utilisateurs ;
- les interactions entre composants graphiques.

## Fonctionnalités

- Addition
- Soustraction
- Multiplication
- Division
- Interface graphique simple

## Technologies utilisées

- Ruby
- Tk

## Exécution

```bash
ruby calculator.rb
