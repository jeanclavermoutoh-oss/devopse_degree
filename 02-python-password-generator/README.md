# Générateur de mots de passe sécurisé en Python

Ce projet consiste à développer un générateur de mots de passe aléatoires et sécurisés en Python.

Le programme génère des mots de passe robustes contenant :

- des lettres minuscules ;
- des lettres majuscules ;
- des chiffres ;
- des caractères spéciaux.

## Objectifs pédagogiques

- Manipuler les chaînes de caractères en Python
- Utiliser les bibliothèques standards
- Comprendre les bases de la sécurité informatique
- Produire des mots de passe aléatoires

## Technologies utilisées

- Python 3

## Exécution

```bash
python password_generator.py
