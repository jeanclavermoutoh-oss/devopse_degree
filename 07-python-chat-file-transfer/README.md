# Application de chat et partage de fichiers

Ce projet académique permet d'apprendre le développement réseau avec Python, SQLite3 et les sockets.

L'objectif principal est de comprendre le fonctionnement des communications temps réel entre plusieurs utilisateurs.

## Fonctionnalités prévues

- Messagerie instantanée
- Communication client/serveur
- Historique des messages SQLite3
- Partage de fichiers
- Interface utilisateur conviviale

## Technologies utilisées

- Python 3
- SQLite3
- Sockets TCP
- Threading

## Exécution du serveur

```bash
python server.py


## Exécution du client

```bash
python client.py
